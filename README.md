# AGENTPRI

AI agent built to automate professional workflows — research, summarization, task routing, and decision support.

Built by Pri Hidalgo as part of an ongoing series of practical AI experiments. This is a working prototype, not a demo — it runs real tasks.

---

## What it does

- Receives a task or question via input
- Routes it to the right AI tool or workflow
- Returns a structured, actionable output
